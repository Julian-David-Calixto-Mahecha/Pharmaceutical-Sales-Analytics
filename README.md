# Pharmaceutical Sales Analytics Dashboard

## Overview

End-to-end Business Intelligence project developed using SQL, Power Query, DAX, and Power BI to analyze pharmaceutical sales performance, budget compliance, and product launch trends.

The project includes data extraction, transformation, dimensional modeling, KPI development, and dashboard visualization.

---

## Technologies Used

- SQL
- Power BI
- Power Query
- DAX
- Microsoft Excel

---

## Project Workflow

### 1. Data Preparation

- Data type standardization
- Null value handling
- Column renaming
- Data validation

### 2. Dimensional Modeling

- Creation of dimension tables using SQL
- Primary and foreign key implementation
- Star schema design

### 3. KPI Development

Main metrics developed using DAX:

- YTD Sales
- YTD Budget
- Compliance Rate
- Freshness Index

### 4. Dashboard Development

Interactive dashboard with filtering capabilities by:

- Country
- Client
- Sales Channel
